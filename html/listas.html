Listas Ordenadas

Atributo das listas

  1. Reversed - Indica que irá ter uma lista inversa
  2. start - Indica por qual elemento deve começar a numeração start = "10"
  3. type
    • 'a' - Indica letras minúsculas
    • 'A' - Indica letras maiúsculas
    • 'i' - Indica Algarimos romanos minúsculos
    • 'I' - Indica Alagarimos romanos maiúsculos
    • '1' - Indica numeros(padrão)
